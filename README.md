#  Cricket Chat Bot  
**AI-Powered Cricket Assistant | React + TypeScript + Google Generative AI**

---

##  Overview
**Cricket Chat Bot** is an **AI-driven conversational web app** that allows users to ask cricket-related questions — from player stats to live match insights — and get instant, natural-language responses.  
Built with **React (TypeScript)** and integrated with **Google Generative AI (Gemini SDK)**, the project demonstrates strong full-stack skills in **frontend architecture, API integration, and AI-powered interaction design**.

---

##  Key Features
-  **Conversational Chat Interface** — Seamless dialogue between user and AI model.  
-  **Google Generative AI SDK Integration** — Enables intelligent cricket discussions and contextual replies.  
-  **Modern UI with Material-UI (MUI)** — Clean, responsive design built with Emotion CSS-in-JS.  
-  **TypeScript-Based Architecture** — Strong type safety, modular components, and scalable design.  
-  **Progressive Web App (PWA)** — Installable app with manifest configuration and offline readiness.  
-  **Unit Testing** — React Testing Library and Jest ensure reliability and performance.
